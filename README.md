# bytechor

## Hackathon

### Install dependencies

* Install miniconda and create python3.12 environment

```shell
conda create --name py312 python=3.12
conda activate py312
```

* Ollama

```shell
ollama pull nomic-embed-text
ollama pull mistral
ollama list
```

* Poetry
```shell
brew install poetry
```

* Poppler - PDF Reading

```shell
brew install poppler
```

* python modules

```shell
poetry install
```

* run

```shell
python -m aichatbot
```
